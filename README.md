Showcase Autosales

Is a website for listing cars of different kind for sale Inspired from [Mekina.net](https://www.mekina.net/)

Features Include

- A home page where cars are displayed and a dummy catagories section where each car catagory links just a Youtube video of the car type.
- Used flexbox to layout the differtent car catagories and also the car listings
